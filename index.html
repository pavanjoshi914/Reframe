Reframe dev server

This is the Vite dev server for an Electron app. The UI runs inside an Electron window — opening this URL in a browser won't show anything useful.

If the Electron window didn't open automatically, check:

  • ELECTRON_RUN_AS_NODE is unset in your shell (env | grep ELECTRON)
  • You ran npm run dev from the project root
  • The HUD pill is small (~620×60). Look near the top-center of your primary display.

The renderer entry points (loaded by Electron) are: /hud.html, /picker.html, /editor.html.
